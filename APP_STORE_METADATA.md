## Description
RummyIQ is the ultimate companion for Indian Rummy players. Track scores effortlessly across all popular game variants including Pool Rummy (101, 201, 250), Deals Rummy, and Points Rummy.

**Key Features:**

- **Multiple Game Variants**: Support for Pool Rummy, Deals Rummy, and Points Rummy
- **Easy Score Entry**: Quick tap-based scoring - mark winner with one tap, cycle through drop/custom scores for others
- **Automatic Winner Detection**: Pool games automatically declare the winner when one player remains
- **Score Progression Chart**: Visual line graph showing how each player's score evolved over rounds
- **Game History**: Unlimited game history with detailed round-by-round breakdown
- **Name Your Games**: Add custom names to games for easy reference
- **Celebration**: Enjoy animated fireworks when a game ends

**Perfect for:**
- Family game nights
- Friendly rummy sessions
- Tournament scoring
- Keeping accurate records

No more pen and paper! Let RummyIQ handle the math while you focus on the game.
